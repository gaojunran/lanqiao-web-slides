# 蓝桥杯Web组备赛指南 · 2025年省赛

## 幻灯片

```bash
# 预先安装node、pnpm、just
pnpm install

just dev guide  # 这将启动slides/guide.md
just dev another-slide  # 这将启动slides/another-slide.md

just pick # 启动交互式选择器，需要预先安装fzf并在bash/zsh中运行
```
